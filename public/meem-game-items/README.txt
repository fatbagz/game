GAME ASSETS FOLDER
==================

Place your meme-themed game assets in this folder.

The game will automatically load and use these assets.
If an asset is missing, the game will use a fallback placeholder.

ASSET NAMING CONVENTION:
------------------------

Player Sprites:
- player-idle.png
- player-run.png
- player-jump.png
- player-fall.png
- player-hurt.png
- player-win.png

Collectibles (numbered 1-5):
- collectible-1.png
- collectible-2.png
- collectible-3.png
- etc.

Enemies (numbered 1-5):
- enemy-1.png
- enemy-2.png
- enemy-3.png
- etc.

Backgrounds (numbered 1-3):
- background-1.png
- background-2.png
- background-3.png

Platforms (numbered 1-3):
- platform-1.png
- platform-2.png
- platform-3.png

RECOMMENDED SIZES:
------------------
- Player: 64x64 pixels
- Enemies: 48x48 pixels
- Collectibles: 32x32 pixels
- Backgrounds: 800x600 pixels or larger
- Platforms: Variable width, 40px height

The game supports PNG format with transparency.
All assets are optional - the game will work with fallback graphics.
